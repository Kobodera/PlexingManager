Installation Guide

This guide is designed to allow you to set the plexing manager up. It will require you to have at least some knowledge of setting up a web server. I personally use the Aprelium Abyss web server and will add a link to it in this document. You will have to set it up yourself thought which is not that hard.

SQL Server setup

Install SQL server express on the intended server. Unless you use a domain I would suggest setting it up as Mixed Mode so that it can handle both windows authentication but also username password.

The default version of the plexingmanager required a user PlexManager, but if you wish to use a diffrent user that is up to you.

  1. Create a new Database. By default this should be called PlexManager, but the name is up to you.
  2. Create a new user, PlexManager and give that user read/write permissions to the PlexManager database.
  3. Run the PlexManagerDB.sql script on the PlexManager database to set the tables up.
  4. Run the BasicData.sql script on the PlexManager database to add basic plex data to the database.
